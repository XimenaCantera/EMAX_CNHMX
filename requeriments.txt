Lista de todas las librerías de Python utilizadas y su justificación:

1. Flask: Permite levantar de manera rápida y con poco consumo de recursos el servidor HTTP para recibir los archivos del frontend de React.

2. Flask-CORS: Permitie que el frontend de React envíe solicitudes y reciba respuestas del backend sin ser bloqueado por el navegador.

3. Pandas: Se usa para manipular los datos y leer los archivos de Excel.

4. Openpyxl: Librería para poder abrir, analizar e interpretar la estructura y formato de los .xlsx cargados al sistema.

5. Dash: Permite construir los gráficos en Python y conecta de forma fluida con los datos procesados por Pandas.

6. Plotly: Motor de visualización que utiliza Dash. Permite generar los gráficos interactivos.

7. Scikit-learn: Librería de machine learning usada para el modelo de Clustering K-Means (StandardScaler + KMeans) que segmenta las unidades en 3 grupos de riesgo operativo.
